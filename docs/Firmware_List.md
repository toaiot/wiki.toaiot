### Firmware Naming Convention

Format: `BMCU_[Version]_[Model]_[Filament_Return_Distance].bin`

Example: `BMCU_27-5_A_filamentreturn_distance_15cm.bin`

- **Version**: BMCU 27-5 or BMCU 370C
- **Model**: A, B, C, or D (printer position)
- **Filament Return Distance**: 15cm to 75cm
