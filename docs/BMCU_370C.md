# BMCU 370C (Hall Sensor Version)

## Product Overview

The BMCU 370C is an advanced version of the Bidirectional Buffer Module, using Hall sensor technology for contactless filament detection. This provides more precise detection without physical contact with the filament.

## Compatible Printers

- Bambu Lab P1P
- Bambu Lab P1S
- Bambu Lab X1C
- Bambu Lab X1E
